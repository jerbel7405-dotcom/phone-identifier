# 📞 Phone Identifier

Logiciel portable pour identifier n'importe quel numéro de téléphone.

## Télécharger le .exe
1. Va dans l'onglet **Actions** de ce repo
2. Clique sur le dernier workflow **Build Windows EXE**
3. Télécharge **PhoneIdentifier-Windows**
4. Lance `PhoneIdentifier.exe` — aucune installation requise !
